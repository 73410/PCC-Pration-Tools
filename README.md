# PCC-Pration-Tools
一款简洁先进的服务器管理工具
# 一、我们能做什么
- 发布各种公告​
- 简单签到​
- 服务器投票​
- 服务器答题​
- ChatGPT聊天​
- 服务器关机倒计时​
# 二、插件指令
- /pcc：插件主指令
- /gonggao：发布公告
- /shout：喊话
- /stopp：倒计时关服
- /vote: 服务器投票
- /question: 发布问题
- /answer: 回答问题
- /chat：ChatGPT聊天
- /signin：签到
- /signin_week：领取连续签到奖励
# 三、权限节点
- pcc.comman.gonggao：插件主指令
- pcc.command.gonggao：发布公告
- pcc.command.shout：喊话
- pcc.command.stop：倒计时关服
- pcc.command.chat：ChatGPT聊天权限
- pcc.sign：签到权限
- pcc.signin_week：领取连续签到奖励权限
